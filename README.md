# Scrabble Web App

## Definition

Online Scrabble for 2 to 4 players

(Frontend Repository)

## Tools and Techs

- React for Frontend
- Python with Django for Backend
- MongoDB for Database
- Cypress for TTD and E2E Testing
- Antd for react components library
